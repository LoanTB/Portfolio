GRAVITY SUIKA GAME

2024

COMPETENCES:

Python

Pygame

Json

SOURCES:

  • LaSource

Jeu inspiré de Suika Game, un jeu de fruits se combinant quand ils ont la même taille et dont le but est d'accumuler le plus de fruits dans un panier sans en faire tomber. Dans ce projet, j'ai imaginé un mode de jeu où le panier est remplacé par une planète, le but étant de ne pas atteindre une trop grande altitude. Ce mode de jeu rend le jeu plus intéressant et divertissant, en plus de la possibilité de modifier le jeu et d'avoir un tableau des scores de tous les joueurs dans toute version personnalisée.
